Plastic Object Detection

Project Overview

Developed a machine learning model to detect plastic objects in images, improving environmental monitoring. Utilized convolutional neural networks (CNNs) for image classification and detection, and built a user-friendly dashboard for visualizing results. The project aimed to assist environmental agencies in tracking plastic waste and automating data collection from field images.

Key Features

  • Automated detection of plastic objects in diverse environments
  • High-accuracy CNN model trained on a custom dataset
  • Interactive dashboard for result visualization
  • Exportable reports for environmental analysis

Technologies Used

  • Python, TensorFlow, Keras
  • OpenCV for image processing
  • Flask for backend API
  • React.js for dashboard UI
