Online Grocery Store

Online Grocery Store is a full-stack e-commerce system designed to support product browsing, cart management, and order processing with strong emphasis on backend data consistency and transactional reliability.

  • Domain: E-commerce Platforms
  • Architecture: Server-Rendered Web Application
  • Tech Stack: Java, JSP, JDBC, MySQL, Bootstrap

Problem → Solution → Impact

  • Problem: Managing real-time product availability, cart state, and user sessions in a multi-user shopping environment while maintaining data consistency.
  • Solution: Designed a backend-driven shopping workflow using JDBC-based data access layers, server-side JSP rendering, and transactional MySQL operations to ensure consistent cart and inventory state.
  • Impact: Delivered a reliable shopping experience with accurate cart updates, reduced data anomalies, and smooth checkout flow.

System Design & Engineering

  • Designed normalized MySQL schemas for products, users, carts, and orders.
  • Implemented JDBC-based DAO layer to isolate database access logic.
  • Ensured transactional consistency during cart updates and order placement.
  • Used server-side rendering (JSP) to reduce frontend complexity and centralize business logic.
  • Handled concurrent user interactions through session management.

Engineering Focus

  • Backend correctness over UI complexity.
  • Clear separation of presentation, business logic, and data layers.
  • Foundational understanding of e-commerce system design.
